## Operating Model

```mermaid
flowchart LR
    A["Messy input"] --> B["Collect"]
    B --> C["Validate"]
    C --> D["Store"]
    D --> E["Model / analyse / rank"]
    E --> F["User-facing app"]
    F --> G["Feedback + tests"]
    G --> C
```

The goal is simple: make the input clear, make the process repeatable, expose the result through something useful, and make failures visible.
