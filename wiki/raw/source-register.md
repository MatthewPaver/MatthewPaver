# Source Register

These are the canonical files the portfolio wiki is compiled from. Treat them as source material; the wiki summarises and connects them.

| Source | Owns |
|:---|:---|
| `README.md` | Public GitHub profile route, featured products, credentials, quality-gate links. |
| `store/app-index.csv` | Catalogue rows, shelves, status, stack, proof, card metadata. |
| `store/previews.json` | Preview-page content, project summaries, links, and notes. |
| `store/index.html` | Visual store layout and card copy. |
| `CASE_STUDIES.md` | Longer project stories, tradeoffs, and private-system explanations. |
| `Projects.md` | Full project appendix and public/private/archive map. |
| `APP_IDEAS.md` | Ranked backlog of portfolio-grade future apps. |
| `PORTFOLIO_AUDIT.md` | Latest deterministic audit output. |
